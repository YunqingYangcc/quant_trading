## 1. 系统能帮你做什么

**TrackQuant** 是面向个人散户的赛道型量化辅助系统。不是全市场海选工具，不做高频交易。

### 一句话定位

> 你选赛道方向（半导体/AI/机器人/存储/材料/电力AI），系统帮你在这个赛道里**排序选股、回测策略、管住风险**。

### 它能做的

| 能力 | 说明 |
|:--|:--|
| 🔍 **赛道内排序** | AI 模型对赛道内每只股票打分排名，告诉你谁强谁弱 |
| 📊 **策略回测** | 20+ 种策略任选，跑一遍历史数据看盈亏，提前知道策略的脾气 |
| 🛡️ **风控约束** | 单票 ≤20%、赛道 ≤50%、涨跌停限制——从架构层面限制最大犯错成本 |
| 📈 **一眼看全局** | K线叠加均线/布林/RSI/ATR，一个页面替代刷五个 APP |
| 📖 **策略说明书** | 每个策略有四维大白话说明：机制原理、适用行情、预期效果、注意事项 |

### 它不能做的

| 不能做 | 原因 |
|:--|:--|
| ❌ 预测明天涨跌 | 没有任何合法系统能做 |
| ❌ 代替你的产业认知 | 赛道方向是你选的，系统只在你选的框架里优化 |
| ❌ 保证赚钱 | AI 准确率约 52%（比瞎蒙高 2%），信号诚实但不强 |

### 当前覆盖赛道

| 赛道 | 股票数 | 说明 |
|:-----|:------:|:-----|
| 半导体 | 51 | 设备/材料/设计/制造/封测 |
| AI | 32 | 算力/光模块/服务器/应用 |
| 机器人 | 25 | 减速器/伺服/控制器/传感器 |
| 存储 | 16 | NAND/NOR/接口/模组 |
| 材料 | 21 | 硅片/光刻胶/靶材/电子化学品 |
| 电力AI | 21 | 电网智能化/AI电源/特高压 |

## 2. 启动系统

### 一键启动

```bash
cd track_quant
./start.sh
```

这个脚本同时启动后端（端口 8000）和前端（端口 5173）。

### 手动启动（两个终端）

**终端 1 — 后端**
```bash
cd backend
uvicorn app.main:app --reload --port 8000
```

**终端 2 — 前端**
```bash
cd frontend
npm run dev
```

### 验证启动成功

| 服务 | 地址 | 正常表现 |
|:-----|:-----|:-----|
| 前端界面 | http://localhost:5173 | 看到深色侧边栏 + Dashboard 首页 |
| 后端 API | http://localhost:8000/health | 返回 `{"status":"healthy"}` |
| API 文档 | http://localhost:8000/docs | Swagger 接口文档（开发调试用） |

## 3. 日常三步法（核心）

> **每天 5 分钟，管住手别乱动。**

### Step 1 — 看全局（30 秒）

打开 Dashboard 首页（`/`），扫一眼赛道景气度：

```
半导体 景气 91 🟢
AI     景气 91 🟢
机器人 景气 91 🟢
存储   景气 91 🟢
材料   景气 90 🟢
电力AI 景气 90 🟢
```

- **景气 > 70（绿色）**：赛道整体健康，正常持仓
- **景气 40-70（黄色）**：中性，观望
- **景气 < 40（红色）**：赛道走弱，检查是不是基本面变了，考虑降仓

### Step 2 — 看排名（1 分钟）

点进你持有的赛道（如半导体 → Alpha Workstation），看左侧 AI 排名：

```
🥇 300604 长川科技  0.498
🥈 688536 思瑞浦    0.488
🥉 688072 拓荆科技  0.487
...
51. xxxxxx         0.452
```

**指导意义**：
- 你已经持有的票，看一眼它排第几。排名稳定的不用动
- 排名从 Top3 突然跌到垫底 → 去 K 线副图看 RSI 是不是超买了、ATR 是不是暴增
- **别盲信排名**——AI 准确率 52%，比瞎蒙高 2 个百分点。作用是「第二意见」，不是神谕

### Step 3 — 跑回测（3 分钟，每周 1-2 次）

打开 Backtest Lab（`/backtest`），选你关心的赛道 + 预设方案，点运行：

```
半导体 · 月频 · Top-3 · 单票≤20%
总收益 +17.8%  |  年化 +5.2%  |  夏普 0.35  |  最大回撤 -39%
```

**指导意义**：
- 回测告诉你这个策略**历史上**的盈亏走势。最大回撤 -39% 意味着你中间会亏近四成——你能扛住吗？
- 扛不住？改参数：Top-3 → Top-5（更分散），加止损 -15%，重新跑看回撤降了多少
- 这个过程让你**提前知道策略的脾气**，而不是真金白银进去才发现吃不消

### 4.4 Alpha Research 因子研究

路径：`/alpha`

### 功能

对因子进行深度分析，验证因子是否可靠：

```
赛道选择 ┌──────────┐  因子选择 ┌──────────┐
         │ 半导体   │          │ rsi_6   │
         └──────────┘          └──────────┘

📊 因子IC时序        → 该因子过去一段时间的预测能力走势
🔥 因子相关性热力图   → Top20 因子之间的相关性（过高的要警惕）
📦 分位数组合收益    → 按因子值分 5 组，看高低组收益差距
📋 因子IC统计表      → Mean IC / IR / IC>0% 比率
```

### 交易员关注点

- **IC（Information Coefficient）**：因子与未来收益的相关性，>0 表示正向预测
- **IR（Information Ratio）**：IC 的稳定性，越大越可靠
- **分位数收益**：Q5(高分) 持续跑赢 Q1(低分) → 因子有效
- 选择某个因子后，可以看到该因子历史 IC 走势

```
模型卡片网格（4 列）
┌──────────┬──────────┬──────────┬──────────┐
│ SEMICON. │   AI     │  ROBOT   │ STORAGE  │
│ ✅ Ready │ ⚠️ Overfit│ ✅ Ready │ ✅ Ready │

│ 0.0512   │ 0.0891   │ 0.0766   │ 0.0623  │

│ MATERIAL │ AI-POWER │          │          │
│ ✅ Ready │ ✅ Ready │          │          │
│ [Retrain]│ [Retrain]│          │          │
└──────────┴──────────┴──────────┴──────────┘

📊 特征重要性 Top10（可切换赛道查看）
📋 模型训练结果比对表（所有模型横向对比）
```

### 操作

1. **查看模型状态**：卡片显示 Train/Val/Test R²、CV Mean R²
2. **一键重训**：点击 `🔄 Retrain` 按钮，系统重新训练该赛道模型
   - 重训过程中按钮显示 "Training..."
   - 完成后自动刷新数据
3. **特征重要性**：选择某个赛道，查看 Top10 特征的贡献度排序

### 交易员关注点

- **Acc（准确率）**：模型预测二分类（跑赢/跑输赛道中位数）的准确率。50% 是随机基线，当前约 50-54%
- **过拟合标记**：Train Acc 和 Val Acc 差距 > 10% 为过拟合，当前全部赛道通过
- **特征重要性**：Top 特征通常是 OBV、VPT、SMA_5 等量价指标

> ⚠️ 注意：页面显示的 R² 字段实际存储的是二分类准确率，不是回归 R²。阅读时理解为「准确率」即可。

### 预设方案说明

| 方案 | 资金 | Top-N | 频率 | 单票上限 | 赛道上限 | 适合场景 |
|:-----|:----:|:-----:|:----:|:--------:|:--------:|:---------|
| 🛡️ 保守 | 5万 | 2 | 月频 | 10% | 30% | 行情不明朗、初次试水 |
| ⚖️ 均衡 | 10万 | 3 | 周频 | 20% | 50% | **推荐**，默认方案 |

### 操作步骤

1. 选择预设方案（或手动调整参数）
2. 点击 **「开始回测」**
3. 等待回测完成（通常 5-30 秒）
4. 查看绩效指标和达标情况

### 关键指标解读

| 指标 | 参考值（半导体 月频 Top-3） | 说明 |
|:-----|:------:|:-----|
| 夏普比率 | 0.35 | 风险调整后收益，>0 即正期望 |
| 最大回撤 | -39% | 历史最大亏损幅度，判断你是否扛得住 |
| 年化收益 | +5.2% | 折合年化收益率 |
| 胜率 | 47% | 交易盈利比例 |

> 注意：验收底线（夏普 ≥ 1.2）是针对机构全市场量化的标准，不适用于 20-50 只股票的小赛道池。散户关注的重点是「最大回撤自己能不能扛」和「收益是否为正期望」。

## 5. 什么时候该动、什么时候不该动

> **散户亏钱的最大原因不是选错股，是管不住手。**

| 场景 | 系统告诉你什么 | 你该做什么 |
|:--|:--|:--|
| 🟢 日常持有 | 打分稳定、景气平稳 | **别动。** 月频调仓就够了，天天看但不要天天操作 |
| 🟡 赛道景气连降 | 景气从 90→70→50 | 检查是不是赛道基本面变了，考虑降仓 |
| 🔴 某只票排名暴跌 | 从 Top3 掉到垫底 | 去 K 线副图看 RSI 是不是超买、ATR 是不是骤增 |
| 🔵 想买新票 | 看看它排第几 | 排前三可以买，排名靠后等一等 |
| ⚪ 纠结要不要卖 | 看看止损 -15% 触发过没 | 没触发就按规则来，别情绪化操作 |
| 🟣 大盘暴跌 | 所有赛道景气骤降 | 别抄底。等景气企稳回升再动 |

### 系统帮你守住的纪律

| 你容易犯的错 | 系统怎么拦你 |
|:--|:--|
| 追涨杀跌 | 策略规定月频调仓，逼你等一个月再动 |
| 一把梭哈 | 单票 ≤20%、赛道 ≤50%，架构层面限制 |
| 死不割肉 | 止损 -15% 自动提醒该走了 |
| 频繁换股 | 回测数据告诉你月频跑赢周频，别天天操作 |

## 6. 定期维护任务

### 每日（手动触发）

打开 Pipeline Runner 工具，点击「更新数据」，拉取最新日线行情。

### 每月（或数据更新后）

1. 打开 Model Factory，点击「一键重训所有模型」（约 10 秒）
2. 打开 Backtest Lab，重跑一次回测看指标是否恶化

### 季度（检查信号质量）

1. 打开 Alpha Research，看 Top 因子的 IC 是否持续衰减
2. 如果主要因子 IC 趋向零，说明市场风格变了，需要关注

## 7. 指标速查表

### 因子指标

| 指标 | 读法 | 参考值 |
|:-----|:-----|:-------|
| **IC** | 因子与未来收益的相关系数。正数=正向因子 | \|IC\|≥0.01 |
| **IR** | IC 均值/标准差，衡量稳定性 | ≥ 0.05 |
| **分组收益** | 按因子值分组，看高低组差距 | 高分持续跑赢低分 |

### 模型指标

| 指标 | 读法 | 当前（半导体） |
|:-----|:-----|:-------|
| **Acc（准确率）** | 二分类预测准确率，50% 是随机基线 | 52-58% |
| **Train/Val Gap** | 训练集 vs 验证集差距 | < 10% ✅ |
| **过拟合** | Train 远好于 Val | 无 ✅ |

### 回测指标

| 指标 | 读法 | 当前（半导体 月频 Top-3） |
|:-----|:-----|:---------|
| **夏普比率** | 风险调整后收益，>0 即正期望 | 0.35 |
| **最大回撤** | 历史最大亏损幅度 | -39% |
| **年化收益** | 折合成年化收益率 | +5.2% |
| **胜率** | 盈利交易占比 | 47% |

> 散户关注重点：① 最大回撤自己能不能扛 ② 收益是否为正期望。机构标准（夏普 ≥ 1.2）不适用于小赛道池。

## 8. 常见问题

### Q: 页面打开是空白的？
A: 确认后端和前端都启动了（终端 1 和终端 2 都在运行）。F5 刷新。

### Q: 数据显示为 0？
A: 检查后端终端是否有报错。可能是数据库需要重建或后端服务未启动。

### Q: Quick Stats 全 0？
A: 后端没连上或者数据库为空。检查 `http://localhost:8000/health` 返回是否正常。

### Q: 点击重训没反应？
A: 重训需要 10-30 秒，期间按钮会显示 "Training..."。如果长时间没返回，检查后端终端是否有错误日志。

### Q: 跑回测报错？
A: 需要模型已训练。先去 Model Factory 页面确认模型状态，如果显示 "模型未训练"，点击 Retrain。

### Q: 模型准确率很低？
A: 准确率约 52-58%（二分类，随机基线 50%）。在小赛道池（20-50 只股票）中，比随机高 2-8 个百分点已经是不错的信号。系统的价值在于排序和风控，不在于精确预测。

### Q: 如何添加新股票？
A: 联系开发人员处理。需要更新 universe.csv、拉取数据、重算特征、重筛因子、重训模型——整套流程。

### Q: K 线图上有些线看不清？
A: 点击图例可以开关每条线。鼠标悬停在 K 线上可以看到该日的详细数据。

### Q: 想查看更长时间范围的 K 线？
A: 底部的 DataZoom 滑块可以拖动选择时间范围，也可以缩放。
